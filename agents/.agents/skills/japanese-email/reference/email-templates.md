# Japanese Business Email Templates

Complete templates for common business scenarios. Copy and customize as needed.

## Meeting Request (お打ち合わせのお願い)

```
件名：【ABC株式会社】お打ち合わせのお願い

田中様

お世話になっております。
ABC株式会社の山田でございます。

先日ご紹介いただいた新サービスについて、
詳しくお話を伺いたく、ご連絡いたしました。

つきましては、下記日程でお時間をいただけないでしょうか。

・10月15日（火）14:00〜
・10月17日（木）10:00〜
・10月18日（金）15:00〜

ご都合の良い日時をお知らせいただけますと幸いです。
何卒よろしくお願い申し上げます。

--
山田太郎
ABC株式会社 営業部
Email: yamada@abc.co.jp
Tel: 03-1234-5678
```

## Apology Email (お詫び)

```
件名：【お詫び】納品遅延について

田中様

お世話になっております。
ABC株式会社の山田でございます。

このたびは、納品が遅れましたこと、
深くお詫び申し上げます。

原因を確認したところ、社内の確認作業に
想定以上の時間がかかってしまいました。

今後はこのようなことがないよう、
工程管理を徹底してまいります。

ご迷惑をおかけし、誠に申し訳ございませんでした。
何卒ご容赦くださいますようお願い申し上げます。

--
山田太郎
```

## Declining a Request (お断り)

```
件名：Re: ご依頼の件について

田中様

お世話になっております。
ABC株式会社の山田でございます。

このたびはご依頼いただき、誠にありがとうございます。

せっかくのお申し出ではございますが、
あいにく現在、他案件の対応で手一杯となっており、
今回はお引き受けすることが難しい状況でございます。

ご期待に沿えず、大変申し訳ございません。
また機会がございましたら、ぜひお声がけいただければ幸いです。

何卒ご理解のほど、よろしくお願い申し上げます。

--
山田太郎
```

これまでの経緯は把握しております。
