# Test: 02-meeting-request-jp (V2 - Japanese)

**Prompt:**
```
田中様に新しいプロジェクトの提案について来週打ち合わせをお願いするビジネスメールを書いてください。
```

**Response:**

田中様宛のビジネスメールを作成いたします。

---

**件名：** 新プロジェクトのご提案に関するお打ち合わせのお願い

---

田中様

いつもお世話になっております。

突然のご連絡失礼いたします。

このたび、新しいプロジェクトのご提案についてぜひご相談させていただきたく、ご連絡申し上げました。

つきましては、来週、お打ち合わせのお時間を頂戴できますでしょうか。

下記の日程でご都合のよろしい日時がございましたら、ご教示いただけますと幸いです。

- 1月27日（月）10:00〜17:00
- 1月28日（火）10:00〜17:00
- 1月29日（水）13:00〜17:00

ご多忙のところ恐れ入りますが、ご検討のほどよろしくお願い申し上げます。

何かご不明な点がございましたら、お気軽にお申し付けください。

どうぞよろしくお願いいたします。

---

[差出人名]
[会社名]
[部署名]
[電話番号]
[メールアドレス]

---

**補足説明：**
- 「いつもお世話になっております」：ビジネスメールの定番の挨拶
- 「ご教示いただけますと幸いです」：相手に情報を求める際の丁寧な表現
- 「ご多忙のところ恐れ入りますが」：相手の時間をいただくことへの配慮
- 候補日を3つ提示することで、相手が選びやすくなります

日程や差出人情報を適宜ご変更ください。
