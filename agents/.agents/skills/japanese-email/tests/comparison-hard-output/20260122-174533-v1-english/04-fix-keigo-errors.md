# Test: 04-fix-keigo-errors (V1 - English)

**Prompt:**
```
この日本語ビジネスメールの敬語の間違いを直してください：

件名：明日の会議
田中さん
お疲れ様です。
明日の会議に私がいらっしゃいます。資料を見てください。
ご飯を食べた後に始めましょう。
よろしく。
山田
```

**Response:**

この日本語ビジネスメールには敬語の間違いがいくつかあります。修正版と解説を示します。

## 修正版

```
件名：明日の会議について

田中様

お疲れ様です。

明日の会議に出席いたします。資料をご確認いただけますでしょうか。
昼食後に開始できればと存じます。

よろしくお願いいたします。

山田
```

## 主な間違いと修正点

| 元の表現 | 問題点 | 修正後 |
|---------|--------|--------|
| 田中さん | ビジネスでは「様」を使う | 田中様 |
| 私がいらっしゃいます | 「いらっしゃる」は尊敬語。自分に使うのは誤り | 出席いたします（謙譲語） |
| 見てください | やや命令的 | ご確認いただけますでしょうか |
| ご飯を食べた後 | カジュアルすぎる | 昼食後 |
| 始めましょう | 提案だが、もう少し丁寧に | 開始できればと存じます |
| よろしく | 省略形で失礼 | よろしくお願いいたします |

## 敬語の基本

- **尊敬語**（相手の動作）：いらっしゃる、ご覧になる、召し上がる
- **謙譲語**（自分の動作）：参る、いたす、申す、拝見する
- **丁寧語**（文末）：です、ます、ございます

「いらっしゃいます」を自分に使ったのが最も大きな間違いです。自分の動作には謙譲語「参ります」「出席いたします」を使います。
