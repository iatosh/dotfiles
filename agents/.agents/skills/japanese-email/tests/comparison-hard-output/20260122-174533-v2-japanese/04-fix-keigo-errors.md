# Test: 04-fix-keigo-errors (V2 - Japanese)

**Prompt:**
```
この日本語ビジネスメールの敬語の間違いを直してください：

件名：明日の会議
田中さん
お疲れ様です。
明日の会議に私がいらっしゃいます。資料を見てください。
ご飯を食べた後に始めましょう。
よろしく。
山田
```

**Response:**

このビジネスメールには敬語の誤りがいくつかあります。修正版と解説をお伝えします。

## 修正版

```
件名：明日の会議について

田中様

お疲れ様です。

明日の会議に出席いたします。資料をご確認ください。
昼食後に開始いたしましょう。

よろしくお願いいたします。

山田
```

## 主な修正点

| 元の文 | 修正後 | 解説 |
|--------|--------|------|
| 田中さん | 田中様 | ビジネスメールでは「様」を使用 |
| 私がいらっしゃいます | 出席いたします | 「いらっしゃる」は尊敬語（相手に使う）。自分には謙譲語「いたします」を使用 |
| 見てください | ご確認ください | より丁寧なビジネス表現 |
| ご飯を食べた | 昼食後 | 「ご飯を食べた」はカジュアルすぎる |
| よろしく。 | よろしくお願いいたします。 | ビジネスメールでは正式な結びの挨拶が必要 |

## 最も重大な誤り

**「私がいらっしゃいます」** - これは敬語の種類を間違えています。
- 尊敬語（相手の動作）：いらっしゃる、ご覧になる
- 謙譲語（自分の動作）：参る、伺う、いたす

自分の動作には謙譲語を使わなければなりません。
