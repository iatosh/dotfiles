# Test: 04-fix-keigo-errors (No Skill)

**Prompt:**
```
この日本語ビジネスメールの敬語の間違いを直してください：

件名：明日の会議
田中さん
お疲れ様です。
明日の会議に私がいらっしゃいます。資料を見てください。
ご飯を食べた後に始めましょう。
よろしく。
山田
```

**Response:**

このビジネスメールには敬語の使い方にいくつかの間違いがあります。修正版と解説をお伝えします。

## 修正版

```
件名：明日の会議について

田中様

お疲れ様です。

明日の会議に出席いたします。資料をご確認ください。
昼食後に開始いたしましょう。

よろしくお願いいたします。

山田
```

## 主な修正点

| 間違い | 修正 | 理由 |
|--------|------|------|
| 田中さん | 田中**様** | ビジネスメールでは「様」を使用 |
| 私が**いらっしゃいます** | **出席いたします** | 「いらっしゃる」は尊敬語（相手に使う）。自分には謙譲語「参る」「出席する」を使う |
| **見てください** | **ご確認ください** | 「見る」より「ご確認」の方がビジネスにふさわしい |
| **ご飯を食べた後** | **昼食後** | 「ご飯を食べる」はカジュアルすぎる |
| **よろしく** | **よろしくお願いいたします** | 省略形はビジネスメールでは失礼 |

## 敬語の基本ルール

- **尊敬語**（相手の動作）：いらっしゃる、ご覧になる、召し上がる
- **謙譲語**（自分の動作）：参る、拝見する、いただく、いたす

今回の最大の間違いは「いらっしゃいます」を自分に使ったことです。これは相手を下げることになってしまいます。
