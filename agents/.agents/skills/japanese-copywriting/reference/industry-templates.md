## E-Commerce Product Description

### Template Structure

```

【商品説明】
[ターゲット]の方におすすめの[商品カテゴリ]です。

[主要ベネフィット1]
[主要ベネフィット2]
[主要ベネフィット3]

【商品詳細】
・サイズ：[寸法]
・素材：[材質]
・重量：[重さ]
・カラー：[色展開]
・生産国：[原産国]

【こんな方におすすめ】
・[ペルソナ1の悩み]でお困りの方

【お客様の声】
「[具体的なレビュー内容]」（[年代][性別][地域]）
```

```

## Ad Copy Templates

### Search Ad (リスティング広告)

```
【見出し1】（15文字以内）
[キーワード]+[ベネフィット]
例：肩こり改善｜初回50%OFF

【見出し2】（15文字以内）
[信頼性/実績]
例：累計10万人が体験

【説明文】（45文字以内）
[ターゲット]向け。[主要ベネフィット]。[CTA]
例：デスクワークの方に。1回で違いを実感。今すぐ予約
```

```
【キャッチ】（10文字以内）

新常識の○○で、[ベネフィット]

## Conversion Rate Optimization

### A/B Test Ideas for Japanese Copy

| Element | Version A | Version B |
|---------|-----------|-----------|
| CTA | 購入する | カートに入れる |
| CTA | 無料で試す | まずはお試し |
| CTA | 今すぐ申し込む | 詳しく見る |
| Headline | 数字なし | 数字あり |
| Headline | ベネフィット訴求 | 課題訴求 |
| Trust | 「人気No.1」 | 「累計○○万個」 |

### High-Converting CTA Phrases

```
最も効果的：
・無料で資料ダウンロード
・まずは30日間お試し（無料）
・3分で簡単お申し込み

効果的：
・詳しい情報を見る
・お見積もりを依頼する
・専門スタッフに相談する

避けるべき：
・今すぐ購入！
・買う
・申し込む（単独では弱い）
```
