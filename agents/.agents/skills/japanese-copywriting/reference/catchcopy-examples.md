## Pattern Analysis

### Pattern 1: Rhythm & Rhyme (韻)

```
やめられない、とまらない かっぱえびせん
(Kappa Ebisen)

キンチョーの夏、日本の夏
(Kincho)

ピッカピカの一年生
(Shogakukan)
```

**Why it works:** Rhythmic patterns stick in memory. The brain processes rhythmic text faster.

### Pattern 2: Specific Numbers

```
ひとつぶ300メートル (Glico)
→ Specific claim creates curiosity and memorability

100人乗っても大丈夫 (Inaba Storage)
→ Concrete demonstration of quality

24時間働けますか (Regain)
→ Provocative question with number
```

**Why it works:** Specific numbers feel researched and trustworthy. Vague claims feel like marketing speak.

### Pattern 3: Question Format

```
24時間働けますか？ (Regain)
あなたのその疲れ、放っておいていいの？
今のままで、本当にいいの？
```

**Why it works:** Questions engage the reader's mind and create self-reflection.

### Pattern 4: Emotional Wordplay

```
お、ねだん以上。(Nitori)
→ "Oh!" exclamation + "more than the price"

あなたと、コンビに (FamilyMart)
→ "With you" + pun on コンビニ/コンビ

い・ろ・は・す (Coca-Cola water)
→ Traditional Japanese syllabary reference
```

**Why it works:** Wordplay creates delight and memorability while conveying meaning.

### Pattern 5: Aspirational Vision

```
おいしい生活 (Seibu)
→ "Delicious Life" - lifestyle aspiration

水と生きる (Suntory)
→ "Living with water" - philosophy

No Music, No Life (Tower Records)
→ Identity statement
```

**Why it works:** Connects product to larger life meaning and values.

### Pattern 6: Problem → Solution

```
あったらいいなをカタチにする (Kobayashi)
→ "We make your wishes real"

ファイト一発！ (Lipovitan D)
→ Direct energy/solution
```

**Why it works:** Addresses customer pain point immediately.

### Formula Templates

**The Contrast Formula:**
```
[Problem/Old way] → [Solution/New way]

古い常識を、新しい常識へ
面倒なことを、簡単に
```

## Analysis Checklist

When studying catchcopy, ask:

1. **Length** - How many characters? (Usually 10-20)
2. **Sound** - Does it have rhythm? Rhyme?
3. **Emotion** - What feeling does it evoke?
4. **Specificity** - Vague or concrete?
5. **Wordplay** - Any puns or double meanings?
6. **Brand fit** - Does it match brand personality?
7. **Memorability** - Can you recall it easily?
8. **Uniqueness** - Could any competitor use it?
