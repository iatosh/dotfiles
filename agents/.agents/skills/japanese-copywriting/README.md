# Japanese Copywriting Skill

A Claude Code skill for writing compelling marketing copy for Japanese audiences.

## Installation

```bash
npx skills add ronantakizawa/japanese-copywriting
```

Or via Claude Code:

```bash
claude mcp add --from-github ronantakizawa/japanese-copywriting
```

## Overview

Japanese marketing copy requires more than translation—it demands **transcreation**: adapting content to maintain emotional impact while respecting cultural nuances. This skill helps you write copy that resonates with Japanese audiences.

## What This Skill Covers

### Copywriting Types
- **キャッチコピー** - Catchphrases and taglines
- **商品説明** - E-commerce product descriptions
- **ランディングページ** - Landing page copy
- **広告コピー** - Ad copy (search, display, social)
- **B2B/SaaS** - Business-focused content

### Key Concepts
- **Transcreation** over translation
- **Soft CTAs** vs aggressive Western style
- **Trust signals** Japanese consumers expect
- **Emotional triggers** that work in Japan
- **Seasonal marketing** (季節感)

## Cultural Principles

| Western Style | Japanese Style |
|---------------|----------------|
| "Buy now!" | "ご検討ください" |
| Hyperbole | Specific numbers |
| Direct commands | Soft invitations |
| FOMO tactics | Fear of mistakes |
| Price focus | Quality/trust focus |
