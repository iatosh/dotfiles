# 数学記述式答案スタイル

## 基本方針

日本語の数学答案（大学の試験・レポート）の流儀で書く。論文や解説文のスタイルではなく，答案用紙に書くスタイルを徹底する。

## 絶対に守るルール

### やってはいけないこと
- 「期待値の計算」「対称性の利用」「結果」などの小見出しをつけない
- 箇条書きで論理を展開しない（文章と数式で流れを作る）
- 「まず〜します」「次に〜を考えます」のような解説調の書き方をしない
- Markdownの `##` `###` を答案本文中に使わない（設問番号などは除く）

### やること
- 文章と数式を自然につなぎながら論理を展開する
- 式番号は $(1)(2)\ldots$ を文中・式末に埋め込む
- 「これより，〜を得る．$(1)$」のように，導出結果を式番号で参照できるようにする
- 「$(1)$ と〜より，〜となる．」のように前の結果を引用しながら進める
- 最後は「したがって，〜である．$\square$」で締める
- 補足・注意事項は結論の **後** にまとめる

## 文体のテンプレート

```
〜とする．〜は定義より，

[式] ... (1)

ここで，〜であるから，[式] ... (2)

(1) と (2) より，

[式]

を得る．したがって，〜である．$\square$

（補足: 〜）
```

## 例

**悪い例（やってはいけない）:**

> ### 期待値の計算
> 
> まず $x_1$ の期待値を求めます。確率密度関数の対称性を利用すると...

**良い例:**

> $x_1$ の期待値は定義より，
> 
> $$\mathbb{E}[x_1] = \int_{-\infty}^{\infty} x_1 \, p(x_1) \, dx_1 \tag{1}$$
> 
> ここで $p(x_1)$ は $x_1$ について偶関数であり，$x_1$ は奇関数であるから，被積分関数は奇関数となる．これより，
> 
> $$\mathbb{E}[x_1] = 0 \tag{2}$$
> 
> を得る．$x_2$ についても同様の議論により $\mathbb{E}[x_2] = 0$ となる．したがって，$\mathbb{E}[x_1] = \mathbb{E}[x_2] = 0$ である．$\square$

## Σの構造などの前提設定

問題に「自由に想定してよい」とある場合は，最もシンプルな設定を選んで明示してから導出を始める。前提を明示したあとは流れを切らずにそのまま答案を続ける。
