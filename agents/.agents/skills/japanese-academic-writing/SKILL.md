# 日本語学術文書執筆スキル

## 概要

このスキルは、日本語での学術文書（論文、予稿、学位論文、レポート）を執筆する際の原則とチェックリストを提供する。

## リファレンス

| ファイル | 内容 | 使用場面 |
|---------|------|---------|
| [JAPANESE-WRITING.md](references/JAPANESE-WRITING.md) | 日本語文章作法チェックリスト | 文・文章・語句レベルの校正 |
| [PAPER-STRUCTURE.md](references/PAPER-STRUCTURE.md) | 論文構成チェックリスト | 論文各セクションの執筆・校正 |

## クイックリファレンス

### 日本語文章の核心原則

1. **新情報後回しの原則**: 文頭に既知情報、文末に新情報
2. **トップヘビーの原則**: 結論先、理由後
3. **大→小の原則**: 一般論から具体例へ
4. **遡及禁止**: 読み手にページを戻らせない

### 論文構成の核心原則

1. **アブストラクト**: 論文全体の要約（序論の要約ではない）
2. **序論**: 「しかしながら」は1回だけの切り札
3. **方法**: 図は説明より先に引用
4. **結果**: 図任せにせずテキストで傾向を記述
5. **考察**: 参考文献を引用しながら議論

### 即時修正可能な問題（検索パターン）

```
を行う|を行っ|を行い    → 「〜する」に変更
という                  → 削除可能か確認
かを                    → 名詞句に変更
と呼ばれる              → 等価の意味で使用していないか確認
がある．|が存在する      → 主語を前に出す
しかしながら|However    → 2回以上使用していないか確認
近年                    → 具体的な年に変更
```

## 使用方法

### 文書レベルの校正

1. `JAPANESE-WRITING.md`のL1（文章構成）とL2（段落構成）を確認
2. `PAPER-STRUCTURE.md`の該当セクション（P1〜P5）を確認

### 文・語句レベルの校正

1. 検索パターンで機械的に発見可能な問題を修正
2. `JAPANESE-WRITING.md`のL3（文構造）とL4（語句・表記）を確認

### 新規執筆

1. 想定読者を設定（`PAPER-STRUCTURE.md` P0-1参照）
2. 各セクションの原則に従って執筆
3. チェックリストで自己レビュー
