## L1: 文章構成

### L1-1: 新情報後回しの原則

**チェック**: 各文の冒頭に、前文で言及されていない概念が突然登場していないか？

**違反の兆候**: 文と文の繋がりが悪い、読み返さないと理解できない

```
✗ てんかんは，再発性の発作を主な症状とする脳疾患である.
  医師は診断のために，長時間にわたって記録された患者の脳波から...
  ↑「医師」が突然登場

○ てんかんは，再発性の発作を主な症状とする脳疾患である.
  その診断のために医師は，長時間にわたって記録された患者の脳波から...
  ↑「その診断」で前文を受けてから新情報「医師」を導入
```

**修正方法**: 旧情報（既出概念）を文頭に、新情報を文末に移動

---

### L1-2: トップヘビーの原則

**チェック**: 理由や細かい説明が先に来て、結論が最後になっていないか？

**違反の兆候**: 最後まで読まないと何の話か分からない

```
✗ ベースラインに対する相対的な変化を捉えるため，アルファパワーについて，
  ベースラインに対する比を特徴量とした．

○ アルファパワーについて，ベースラインに対する比を特徴量とした．
  これは，ベースラインに対する相対的な変化を捉えるためである．
```

**修正方法**: 「〜ため，〜した」→「〜した．これは〜ためである」

---

### L1-3: 大→小の原則

**チェック**: 具体例・固有名詞が一般的な説明より先に来ていないか？

```
✗ BERT，BART, T5 や GPT などの大規模言語モデルは...
○ 大規模言語モデルである BERT，BART, T5 や GPTは...

✗ リガーレ，ヴィアティン，リオレーナなどといったプロバレーボールチームは...
○ プロバレーボールチームである，リガーレ，ヴィアティン，リオレーナなどは...
```

**修正方法**: 上位概念を先に、具体例を後に

---

### L1-4: 遡及禁止の原則

**チェック**: 読者が前のページに戻らないと理解できない記述がないか？

**違反の兆候**:
- 定義されていない略語の使用
- 「前述の〜」が頻出
- 数式を定義前に引用

**修正方法**: 必要な情報はその場で提供するか、明示的に参照を示す

---

## L2: 段落構成

### L2-1: パラグラフライティング

**チェック**: 各段落の第1文を抜き出して読んだとき、文章の骨格が分かるか？

**原則**: 一段落一テーマ、一節一テーマ

**修正方法**: 段落の主題（トピックセンテンス）を第1文に置く

---

### L2-2: 段落は2文以上

**原則**:
- 段落内の文は2つ以上
- 章内の節は2つ以上
- 節内の小節は2つ以上
- 箇条書きの項目は2つ以上

**修正方法**: 前後の段落と統合するか、内容を補足して2文以上にする

---

## L3: 文構造

### L3-1: 主述の一致

**チェック**: 文中のすべての述語に対応する主語が明確か？

```
✗ 脳波の変化は，1/f^kの変化で表すことができ，音楽聴取における
  スケーリング指数の変化と喜びの関連について調査した
  → 「変化は...調査した」？

○ 脳波の変化は，1/f^kの変化で表すことができる．
  本研究では，音楽聴取におけるスケーリング指数の変化と喜びの関連について調査した．
```

**修正方法**: 文を分割するか、主語を明示する

---

### L3-2: 連用形の濫用

**チェック**: 「〜し」「〜であり」で文を繋いでいる場合、関係性が明確か？

**「〜し」の問題**: 時系列・因果・並列のどれか不明

```
✗ コンビニに行き，おでんを買った．
○ おでんを買うためにコンビニに行った．（因果を明示）
○ コンビニに行った後，おでんを買った．（時系列を明示）
```

**「〜であり」の問題**: A and B か A therefore B か不明

```
曖昧：てんかんは神経疾患であり，認知機能に悪影響を及ぼす．
○ てんかんは神経疾患であり，認知機能に悪影響を及ぼすことが知られている．
```

**A and B 型を明示するには**: AとBの主語を変える

---

### L3-3: 粒度の統一

**チェック**: 「AとB」「A, B, C」で並べる要素の粒度が揃っているか？

**文法的粒度**:
```
✗ サンプル数を増やす，Penalty methods，Early stopping
  → 文 / 名詞 / 名詞

○ サンプル数の増加，Penalty methods，Early stopping
  → すべて名詞
```

**原則**: Aが句ならBも句、Aが節ならBも節で揃える

---

### L3-4: 修飾語の位置

**チェック**: 主語と述語の間に長い修飾語が入っていないか？

```
✗ ECoG のハイガンマ帯がドラムの音を呈示した際に包絡線により変調される

○ ドラムの音が呈示されている間に，その音の包絡線が，ECoGのハイガンマ帯を変調する
```

**修正方法**: 長い修飾語は文頭に移動、または文を分割

---

### L3-5: 受動態の回避

```
✗ 実験参加者は3種類のタスクを課され，ECoGが記録された．
○ 実験参加者には3種類のタスクを課し，ECoGを記録した．
```

---

### L3-6: 接続詞の適切性

**確認すべき点**:
- 「しかし」の後に本当に逆接の内容が来ているか
- 「また」で並列すべき内容か
- 「したがって」の前に根拠があるか

---

### L3-7: 隠れた主語の不一致

**チェック**: 「Aに加えてB」などの構文で、省略された主語・目的語が一致しているか？

```
✗ 大量かつ高品質なアノテーションが付いていて，加えてすでに優れた
  特徴抽出能力を持つImageNetを使った事前学習が，...

分解すると：
  （ImageNetには）大量かつ高品質なアノテーションが付いていて
  加えて
  （事前学習モデルは）すでに優れた特徴抽出能力を持つ
  → 省略された主語が異なる
```

---

### L3-8: 条件の混同

**チェック**: 「PができればQができる」という表現で、十分条件と必要条件を混同していないか？

```
✗ 脳波から感情を推定できれば，より自然なインタフェースが実現できる．

○ より自然なインタフェースの実現には，脳波からの感情推定が必要である．
```

## L4: 語句・表記

### L4-1: 「それぞれ」の挿入

```
✗ AとBは，大文字と小文字である．
○ AとBは，それぞれ大文字と小文字である．
```

---

### L4-2: 「行う」の排除

**検索パターン**: `を行う|を行っ|を行い`

```
✗ 頭蓋内脳波記録を行った → ○ 頭蓋内脳波を記録した
✗ 実験を行う → ○ 実験する
```

---

### L4-3: 間接疑問文の回避

```
✗ 提案手法と既存手法の間に有意差があるかを確認する．
○ 提案手法と既存手法の有意差を確認する．
```

---

### L4-4: 「という」の回避

```
✗ 傾聴，想像，発話するという3種類のタスクを課した
○ 傾聴，想像，発話の3種類のタスクを課した
```

---

### L4-5: 言い換え表現

**問題**: 「Aと呼ばれるB」はA⊂B（AはBの一種）を表すこともある

```
✗ 持続的注意と呼ばれる集中力は，現代において重要な認知機能の一つである．

○ 持続的注意は集中力とも呼ばれ，現代において重要な認知機能の一つである．
```

---

### L4-6: 記号の統一

**原則**: 括弧内のモードで決まる

- 日本語文中で英語を括る → 括弧は全角、中は英語ルール（半角コンマ）
  ```
  ○ 3種類のエントロピー（Approximate, Sample, Permutation）を…
  ```

- 日本語のみ → すべて全角
  ```
  ○ Approximate，Sample，Permutationの3つのエントロピーを…
  ```
